What is panic?

An extensible, open platform to help you manage emergency situations using wearables.

Tested with a Thalmic Myo armband

a simple gesture will save GPS coordinates,record audio and text an emergency contact.

All information is available online to allow for later retrieval and further analytics.

The web interface and backend were built using a MEAN stack.

The Android app was responsible for interfacing with the Myo and for communicating with the API

Developed for Wearhacks 2015

Disclaimer: Panic is not affiliated with, nor is monitored by emergency services. It was built as a proof of concept.
Please refer to your local emergency services in case of emergency.
